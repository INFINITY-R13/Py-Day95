# Py-Day95 - Space Invaders

A Python implementation of the classic Space Invaders arcade game using Pygame.

## Installation

No installation needed! This game uses tkinter which comes built-in with Python.

## How to Play

```bash
python space_invaders.py
```

### Controls
- **Left/Right Arrow Keys**: Move the player ship
- **Spacebar**: Shoot bullets
- **R**: Restart game (after game over)

### Objective
Shoot down all the alien ships before they reach the bottom of the screen. Each alien destroyed gives you 10 points. When you clear all aliens, a new wave appears!

## Features
- Player movement and shooting
- Alien fleet with horizontal movement
- Collision detection
- Score tracking
- Game over condition
- Wave system (new aliens spawn after clearing a wave)
